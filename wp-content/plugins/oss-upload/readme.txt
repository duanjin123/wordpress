== Description ==

使用阿里云 OSS 作为媒体库附件存储空间，支持原创增强的 OSS 协议包装和全原生图像编辑及衍生功能。

### 功能 ###

* 独创协议支持所有原生上传功能
* 支持修改默认的图像编辑器类
* 支持禁止生成多尺寸缩略图
* 带中英文附带设置说明和示范

### 更多 ###

[https://www.xiaomac.com/2016121895.html](https://www.xiaomac.com/2016121895.html)

== Installation ==

1. Upload the plugin folder to the "/wp-content/plugins/" directory of your WordPress site
2. Activate the plugin through the 'Plugins' menu in WordPress
3. See Settings -> OSS Upload
